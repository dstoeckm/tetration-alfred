What is this?

Alfred is a Kafka consumer application that answers Cisco Tetration Analytics "questions".

What's in it for me?

Currently, Alfred supports only one question: asset annotations based on Cisco ACI endpoint tracker.
Coming soon: email notifications sent by Tetration User Apps.

Usage

In the "Configuration" tab, specify your Tetration, Kafka and APIC configuration.
Once done, click on the submit button and confirm. Alfred will start automatically.
The "Operate" tab will provide you a view on the current status as well as controls to alter Alfred services.
You can now move to Tetration and ask questions! Visit Alfred Github page to learn more and find sample user apps.

Warning: Alfred API is either down or unreachable from the browser (TCP/5000)
